# builtin plugins

This directory contains plugins that are embedded in the huxdemp binary, and
always accessible.

The following plugins will eventually be written (PRs welcome!):

- farbfeld viewer
- utf-32
- `num` plugin:
	- f16, f32, f64
	- i8, i16, i32, i64
	- u16, u32, u64
- retroforth disassembler
